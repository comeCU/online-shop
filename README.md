# online-shop

### 我们的目的

练习使用git、maven工具协作开发规模较大的项目，自行处理分支合并冲突等问题。

### 如何启动

1. 克隆项目到本地。

   ```
   git clone https://github.com/comeCU/online-shop.git
   ```

2. 导入开发工具（Eclipse为例）。

   ```
   Import --> Existing Maven Projects
   ```

3. Config Build Path配置成自己jdk版本，在shop-web子模块，启动Tomcat服务器。

4. 开始做你的功能模块吧。
